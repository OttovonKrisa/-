ИНСТРУКЦИЯ ДЛЯ ЗАПУСКА КАРТЫ С ФЛЕШКИ (ОФФЛАЙН РЕЖИМ)
=====================================================

Вы можете запустить это приложение на любом компьютере полностью без интернета,
используя всего ОДИН единственный файл!

КАК ПОЛУЧИТЬ ФАЙЛ ДЛЯ ФЛЕШКИ:
-----------------------------
1. Соберите проект на своем компьютере (или скачайте собранный архив).
   Команда для сборки в терминале: npm run build
2. После сборки в проекте появится папка `dist/`.
   Внутри папки `dist/` будет находиться всего один файл — `index.html`.
3. Скопируйте этот файл `index.html` на вашу флешку.

КАК ЗАПУСКАТЬ НА ДРУГОМ КОМПЬЮТЕРЕ (БЕЗ ИНТЕРНЕТА):
--------------------------------------------------
1. Вставьте флешку в нужный компьютер (интернет не требуется!).
2. Дважды кликните по файлу `index.html` на флешке.
3. Приложение откроется в любом браузере (Chrome, Edge, Firefox, Яндекс.Браузер и др.).
4. Все функции будут работать на 100%:
   - Интерактивная векторная карта (специальный встроенный оффлайн-режим).
   - Точки TPU, Росатома и Совместных проектов.
   - Панель администратора (создание, удаление, импорт точек).
   - Календарь ночного энергосбережения и переходы.
   - Полноэкранный режим.

КАК СЕЙЧАС СКАЧАТЬ ФАЙЛ ИЗ ИНТЕРФЕЙСА:
--------------------------------------
Вы можете зайти в настройки сверху («Settings» в AI Studio), выбрать экспорт в ZIP
или выгрузить папку `dist` напрямую после сборки — в ней лежит полностью автономный файл.

Технические особенности:
- Нет CORS-ошибок при запуск с протокола 'file://' (все скрипты, SVG и стили вшиты прямо в HTML).
- Данные (точки и настройки сна) сохраняются в локальное хранилище браузера (localStorage) каждого компьютера.
- При появлении сети вы сможете выгрузить созданные точки в Firebase Cloud.
